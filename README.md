# BookNest-Backend

Welcome to the BookNest Backend repository! This repository contains all the microservices for the BookNest application.

## Microservices Architecture

The backend is built using a Spring Boot microservices architecture. Each service has been isolated and pushed to its respective feature branch:

- **API Gateway**: `feature/api-gateway`
- **Auth User Service**: `feature/auth-user-service`
- **Cart Service**: `feature/cart-service`
- **Discovery Server**: `feature/discovery-server`
- **Notification Service**: `feature/notification-service`
- **Order Service**: `feature/order-service`
- **Product Service**: `feature/product-service`
- **Review Service**: `feature/review-service`
- **Wishlist Service**: `feature/wishlist-service`

## Getting Started

To work on a specific service, you can check out its feature branch:

```bash
git checkout feature/<service-name>
```

For more details on each service, please refer to the individual branch implementations.
